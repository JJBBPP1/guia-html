Cómo empezar en HTML

Navegación
Guía HTML Guía CSS

HTML (HyperText Markup Language) es el lenguaje que usamos para crear la estructura de una página web. Con él podemos definir títulos, párrafos, imágenes, enlaces ¡y mucho más!

1. Lo que necesitas

  • Un editor de texto: VS Code, Sublime Text, Atom o incluso el bloc de notas.
  • Un navegador web: Chrome, Firefox, Edge o el que prefieras.

2. Crear tu primer archivo

  1. Abre tu editor de texto.
  2. Crea un archivo nuevo y guárdalo como index.html.

3. Estructura básica de HTML

Copia este código en tu archivo:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primera página</title>
</head>
<body>
  <h1>¡Hola, mundo!</h1>
  <p>Esta es mi primera página en HTML.</p>
</body>
</html>

4. Ver tu página

Guarda el archivo, haz doble clic sobre index.html y se abrirá en tu navegador. ¡Ya creaste tu primera página web!

5. Próximos pasos

  • Añadir imágenes con <img>.
  • Crear enlaces con <a>.
  • Hacer listas con <ul> y <ol>.
  • Dar estilos básicos con CSS.

6. Cómo agregar imágenes

Para mostrar una imagen usa la etiqueta <img>:

<img src="ruta/a/tu-imagen.jpg" alt="Descripción de la imagen">

7. Cómo crear enlaces

Para crear un enlace utiliza la etiqueta <a>:

<a href="https://www.ejemplo.com">Visita Ejemplo</a>

8. Cómo hacer listas

Las listas pueden ser ordenadas (<ol>) o desordenadas (<ul>):

<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

<ol>
  <li>Primero</li>
  <li>Segundo</li>
</ol>

9. Cómo dar estilos con CSS

Puedes agregar estilos usando la etiqueta <style> dentro del <head>:

<style>
  body {
    background: #f4f6f8;
    color: #333;
  }
</style>

10. Cómo crear tablas

Las tablas se crean con <table>, <tr> (fila) y <td> (celda):

<table>
  <tr>
    <td>Nombre</td>
    <td>Edad</td>
  </tr>
  <tr>
    <td>Ana</td>
    <td>23</td>
  </tr>
</table>

11. Cómo usar comentarios

Los comentarios en HTML no se muestran en la página:

<!-- Esto es un comentario -->

12. Cómo estructurar formularios

Los formularios permiten recopilar datos del usuario:

<form>
  <label>Nombre:</label>
  <input type="text" name="nombre">
  <button type="submit">Enviar</button>
</form>

13. Cómo insertar videos

Puedes insertar videos con la etiqueta <video>:

<video src="video.mp4" controls>Tu navegador no soporta el video.</video>

14. Cómo usar favicon

El favicon es el ícono que aparece en la pestaña del navegador:

<link rel="icon" href="favicon.ico">

15. Recursos para aprender más

  • MDN Web Docs: HTML
  • W3Schools: HTML Tutorial
  • HTML Reference

16. Cómo insertar audio

Puedes insertar audio con la etiqueta <audio>:

<audio src="audio.mp3" controls>Tu navegador no soporta el audio.</audio>

17. Cómo usar etiquetas semánticas

Las etiquetas semánticas ayudan a estructurar mejor tu página:

<header>Encabezado</header>
<nav>Menú de navegación</nav>
<main>Contenido principal</main>
<footer>Pie de página</footer>

18. Cómo crear botones personalizados

Usa la etiqueta <button> y personalízala con CSS:

<button>Haz clic aquí</button>
<style>
  button {
    background: #2563eb;
    color: #fff;
    border: none;
    padding: 0.7rem 1.5rem;
    border-radius: 8px;
    cursor: pointer;
    font-size: 1rem;
  }
</style>

19. Cómo usar fuentes externas

Puedes usar Google Fonts para mejorar la tipografía:

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
<style>
  body {
    font-family: 'Roboto', sans-serif;
  }
</style>

20. Cómo hacer responsive tu página

Agrega la meta etiqueta viewport y usa media queries en CSS:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
  @media (max-width: 600px) {
    body {
      padding: 1rem;
    }
    #como-empezar {
      padding: 1rem;
    }
  }
</style>

21. Cómo usar imágenes SVG

Las imágenes SVG se insertan igual que otras imágenes, pero permiten gráficos escalables:

<img src="logo.svg" alt="Logo en SVG">
<!-- O directamente en el HTML -->
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="blue" stroke-width="4" fill="lightblue" />
</svg>

22. Cómo crear menús de navegación

Un menú de navegación básico se hace con listas y enlaces:

<nav>
  <ul>
    <li><a href="#inicio">Inicio</a></li>
    <li><a href="#servicios">Servicios</a></li>
    <li><a href="#contacto">Contacto</a></li>
  </ul>
</nav>

23. Cómo agregar animaciones con CSS

Las animaciones se logran con la propiedad animation:

<style>
  .animado {
    animation: girar 2s infinite linear;
  }
  @keyframes girar {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
  }
</style>
<div class="animado">Animación CSS</div>

24. Cómo validar formularios en HTML

HTML permite validaciones básicas con atributos:

<form>
  <input type="email" required placeholder="Correo electrónico">
  <button type="submit">Enviar</button>
</form>

25. Cómo usar atributos data-* personalizados

Los atributos data-* permiten guardar información extra en los elementos:

<div data-usuario="Juan">Hola, Juan</div>
<!-- Se accede con JavaScript: -->
<script>
  const div = document.querySelector('[data-usuario]');
  alert(div.dataset.usuario);
</script>

26. Cómo usar el atributo alt en imágenes

El atributo alt describe la imagen para accesibilidad y SEO:

<img src="foto.jpg" alt="Foto de perfil">

27. Cómo usar el atributo title

El atributo title muestra un tooltip al pasar el mouse:

<a href="#" title="Ir al inicio">Inicio</a>

28. Cómo usar el atributo tabindex

Controla el orden de tabulación en los elementos:

<input type="text" tabindex="1">

29. Cómo usar el atributo aria-label

Mejora la accesibilidad describiendo elementos:

<button aria-label="Cerrar ventana">X</button>

30. Cómo usar el atributo rel en enlaces

Define la relación entre el documento y el enlace:

<a href="https://ejemplo.com" rel="noopener">Abrir Ejemplo</a>

31. Cómo usar el atributo target

Abre enlaces en una nueva pestaña:

<a href="https://ejemplo.com" target="_blank">Abrir Ejemplo</a>

32. Cómo usar el atributo download

Permite descargar archivos desde un enlace:

<a href="archivo.pdf" download>Descargar PDF</a>

33. Cómo usar el atributo placeholder

Muestra texto de ayuda en campos de formulario:

<input type="text" placeholder="Escribe tu nombre">

34. Cómo usar el atributo required

Hace obligatorio un campo de formulario:

<input type="email" required>

35. Cómo usar el atributo readonly

Hace que un campo no se pueda modificar:

<input type="text" value="Solo lectura" readonly>

36. Cómo usar el atributo disabled

Desactiva un campo o botón:

<button disabled>No disponible</button>

37. Cómo usar el atributo autofocus

Enfoca automáticamente un campo al cargar la página:

<input type="text" autofocus>

38. Cómo usar el atributo maxlength

Limita la cantidad de caracteres en un campo:

<input type="text" maxlength="10">

39. Cómo usar el atributo pattern

Valida el formato de entrada con una expresión regular:

<input type="text" pattern="[A-Za-z]{3,}">

40. Cómo usar el atributo value

Define el valor inicial de un campo:

<input type="text" value="Ejemplo">

41. Cómo usar el atributo min y max

Limita el rango de valores numéricos:

<input type="number" min="1" max="10">

42. Cómo usar el atributo step

Define el incremento de los valores numéricos:

<input type="number" step="0.5">

43. Cómo usar el atributo multiple

Permite seleccionar varios archivos o valores:

<input type="file" multiple>

44. Cómo usar el atributo checked

Marca una casilla o radio por defecto:

<input type="checkbox" checked>

45. Cómo usar el atributo selected

Selecciona una opción por defecto en un select:

<option selected>Opción 1</option>

46. Cómo usar el atributo name

Identifica los campos de formulario para el envío:

<input type="text" name="usuario">

47. Cómo usar el atributo id

Identifica elementos únicos en la página:

<div id="principal">Contenido</div>

48. Cómo usar el atributo class

Agrupa elementos para aplicar estilos o scripts:

<div class="caja">Caja</div>

49. Cómo usar el atributo style

Aplica estilos en línea a un elemento:

<p style="color: green;">Texto verde</p>

50. Cómo usar el atributo lang

Define el idioma del contenido:

<html lang="es">

51. Cómo usar el atributo charset

Define la codificación de caracteres:

<meta charset="UTF-8">

52. Cómo usar el atributo defer en scripts

Retrasa la ejecución de scripts hasta que el HTML esté listo:

<script src="app.js" defer></script>

53. Cómo usar el atributo async en scripts

Ejecuta scripts de forma asíncrona:

<script src="app.js" async></script>

54. Cómo usar el atributo src en scripts e imágenes

Define la fuente de archivos externos:

<script src="app.js"></script>
<img src="foto.jpg">

55. Cómo usar el atributo href en enlaces y hojas de estilo

Define la dirección de destino:

<a href="https://ejemplo.com">Enlace</a>
<link href="estilos.css" rel="stylesheet">

56. Cómo usar el atributo type

Define el tipo de elemento o archivo:

<input type="password">
<script type="module" src="main.js"></script>

57. Cómo usar el atributo for en etiquetas label

Asocia una etiqueta label con un campo de formulario:

<label for="email">Correo:</label>
<input id="email" type="email">

58. Cómo usar el atributo action en formularios

Define la URL a la que se envían los datos:

<form action="procesar.php">...</form>

59. Cómo usar el atributo method en formularios

Define el método de envío (GET o POST):

<form method="post">...</form>

60. Cómo usar el atributo enctype en formularios

Define el tipo de codificación para archivos:

<form enctype="multipart/form-data">...</form>

61. Cómo usar el atributo rows y cols en textarea

Define el tamaño de un área de texto:

<textarea rows="4" cols="50"></textarea>

62. Cómo usar el atributo wrap en textarea

Controla el ajuste de texto en textarea:

<textarea wrap="hard"></textarea>

63. Cómo usar el atributo spellcheck

Activa la revisión ortográfica en campos de texto:

<textarea spellcheck="true"></textarea>

64. Cómo usar el atributo autocomplete

Activa o desactiva el autocompletado en formularios:

<input type="text" autocomplete="off">

65. Cómo usar el atributo list en campos de texto

Permite sugerencias con datalist:

<input list="frutas">
<datalist id="frutas">
  <option value="Manzana">
  <option value="Banana">
</datalist>

66. Cómo usar el atributo accept en input file

Restringe los tipos de archivos que se pueden subir:

<input type="file" accept="image/*">

67. Cómo usar el atributo draggable

Permite arrastrar elementos en la página:

<div draggable="true">Arrástrame</div>

68. Cómo usar el atributo contenteditable

Permite editar el contenido de un elemento:

<div contenteditable="true">Texto editable</div>

69. Cómo usar el atributo hidden

Oculta elementos en la página:

<div hidden>No se muestra</div>

70. Cómo usar el atributo data-* para JavaScript

Guarda información personalizada para scripts:

<div data-rol="admin">Administrador</div>
<script>
  const rol = document.querySelector('[data-rol]').dataset.rol;
  console.log(rol);
</script>
